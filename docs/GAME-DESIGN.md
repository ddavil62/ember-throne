## 1. 전투 시스템

### 1.1 기본 구조

그리드 기반 턴제 전투. **플레이어 페이즈 → 적 페이즈** 순서로 진행된다.

- 플레이어 페이즈: 아군 전 유닛을 자유 순서로 이동/행동할 수 있다. 모든 유닛의 행동이 끝나거나 "페이즈 종료"를 선택하면 적 페이즈로 넘어간다.
- 적 페이즈: 적 AI가 모든 적 유닛을 순차 행동시킨다.
- NPC 페이즈: 호위 대상, 중립 유닛이 존재하는 맵에서만 발생한다.

### 1.2 유닛 행동

각 유닛은 턴당 **이동 1회 + 행동 1회**를 수행할 수 있다.

| 행동 | 설명 |
|------|------|
| 공격 | 무기 사거리 내 적에게 물리 공격 |
| 스킬 | MP를 소모하여 고유 스킬 사용 |
| 아이템 | 소비 아이템 사용 (자신 또는 인접 아군) |
| 대기 | 행동 없이 턴 종료. 방어 보정 +10% |
| 방어 | 이동 포기 시 선택 가능. 받는 피해 -30%, 방어 보정 +20% |

- 이동 후 행동, 행동 후 이동 모두 가능하다 (단, 각 1회씩).
- "이동만 하고 행동 안 함"도 가능하다 (남은 행동은 소멸).

### 1.3 이동 범위와 공격 범위

- **이동 범위**: 유닛의 MOV 스탯에 의해 결정. 맨해튼 거리(다이아몬드 형태) 기준.
- **공격 범위**: 무기 유형에 따라 다름.

| 무기 유형 | 기본 사거리 | 비고 |
|-----------|------------|------|
| 검/도끼/창 | 1칸 | 인접 4방향 |
| 창(장창) | 1~2칸 | 관통 불가 |
| 활 | 2~3칸 | 인접 불가 (최소 사거리 2) |
| 마법 | 1~3칸 (스킬별 상이) | 스킬마다 범위/형태 다름 |
| 단검 | 1칸 | 인접 4방향, 후방 공격 시 치명타 보정 |
| 지팡이(타격) | 1칸 | 긴급용, 낮은 위력 |

### 1.4 지형 효과

맵의 각 타일에는 지형 유형이 지정되며, 유닛에게 영향을 준다.

| 지형 | 이동 비용 | 회피 보정 | 방어 보정 | 특수 효과 |
|------|-----------|-----------|-----------|-----------|
| 평지 | 1 | 0% | 0% | - |
| 숲 | 2 | +15% | +5% | 활 사거리 -1 |
| 산/바위 | 3 | +10% | +10% | 중장 유닛 진입 불가 |
| 물(얕은) | 2 | -5% | -5% | 마법 공격 +10% |
| 물(깊은) | 진입 불가 | - | - | 특수 스킬로만 이동 |
| 성벽/건물 | 진입 불가 | - | - | 파괴 가능한 건물 존재 |
| 폐허 | 1 | +5% | 0% | - |
| 모래 | 2 | 0% | 0% | 기마 유닛 MOV -1 |
| 다리 | 1 | 0% | 0% | 파괴 가능 |
| 결계석 | 1 | 0% | 0% | 인접 아군 턴 시작 시 HP 5% 회복 |
| 재의 땅 | 2 | 0% | -10% | 턴 시작 시 HP 5% 감소 |

### 1.5 높이 시스템

본작은 **높이차 개념을 도입하지 않는다.** 탑다운 2D 그리드를 유지하며, 지형 효과로 전술적 다양성을 확보한다. 산/바위는 높이 대신 "진입 불가 또는 높은 이동 비용"으로 표현한다.

### 1.6 전투 공식

```
물리 데미지 = (공격자 ATK * 스킬 배율) - (방어자 DEF / 2)
마법 데미지 = (공격자 MATK * 스킬 배율) - (방어자 MDEF / 2)
최종 데미지 = 기본 데미지 * 지형 보정 * 상성 보정 * (1 + 치명타 보정)
명중률 = 공격자 명중 - 방어자 회피 - 지형 회피 보정
치명타율 = 기본 치명타율 + (SPD차 / 4)
```

- 최소 데미지: 1 (0 이하 방지)
- 명중률 범위: 5%~99% (확정 명중/회피 없음)
- 치명타 데미지: 기본 데미지 * 1.5

### 1.7 상태이상

| 상태 | 효과 | 지속 | 회복 |
|------|------|------|------|
| 독 | 턴 시작 시 최대 HP 8% 감소 | 3턴 | 해독제, 치유 스킬 |
| 화상 | 턴 시작 시 최대 HP 5% 감소, DEF -15% | 2턴 | 자연 치유, 물 지형 진입 시 즉시 해제 |
| 빙결 | 행동 불가 | 1턴 | 피격 시 해제, 화염 스킬 해제 |
| 마비 | 이동 불가, 행동은 가능 | 2턴 | 회복 아이템, 치유 스킬 |
| 침묵 | 스킬 사용 불가 | 2턴 | 자연 치유, 회복 아이템 |
| 수면 | 행동 불가, 피격 시 해제 | 3턴 (피격 해제) | 피격, 회복 아이템 |
| 재화(灰化) | 턴 시작 시 최대 HP 10% 감소, 모든 스탯 -10% | 3턴 | 결계석 인접 시 즉시 해제, 상급 치유 |

"재화"는 재의 바다 관련 적이 사용하는 고유 상태이상으로, 스토리 후반에 등장한다.

### 1.8 승리/패배 조건

- **기본 승리**: 적 전멸
- **기본 패배**: 카엘(주인공) 전투불능
- **특수 조건**: 맵별로 추가 조건 설정 가능
  - 방어전: N턴 생존
  - 호위전: 특정 NPC 생존
  - 탈출전: 카엘이 탈출 지점 도달
  - 보스전: 보스 유닛 격파 (잡졸 전멸 불필요)

## 2. 클래스 시스템

### 2.1 설계 원칙

- 12인의 플레이어블 캐릭터 각각 고유 클래스를 가진다.
- 동일 클래스는 없다. 캐릭터 = 클래스의 1:1 대응.
- 스토리 특정 시점(주로 3막)에서 전직 이벤트가 발생한다.
- 전직은 자동(스토리 연동)이며, 레벨/경험치 조건은 없다. 캐릭터 성장 서사와 연결된다.

- 자유 편성: 나머지 슬롯

## 3. 성장 시스템

### 3.1 경험치와 레벨

- 최대 레벨: **30**
- 경험치 획득: 전투 행동(공격, 치유, 버프, 아이템 사용) 시 획득
- 적 처치 시 보너스 경험치 (적 레벨과의 차이에 비례 조정)
- 전투 미참가 유닛: 전투 종료 시 참가 유닛 평균 경험치의 50% 획득 (레벨 격차 방지)

### 3.2 스탯

| 스탯 | 약어 | 설명 |
|------|------|------|
| 체력 | HP | 0이 되면 전투불능 |
| 마력 | MP | 스킬 사용에 소모 |
| 공격력 | ATK | 물리 데미지 기반 |
| 방어력 | DEF | 물리 데미지 경감 |
| 마법공격력 | MATK | 마법 데미지 기반 |
| 마법방어력 | MDEF | 마법 데미지 경감 |
| 속도 | SPD | 회피, 치명타, 행동 우선순위에 영향 |
| 이동력 | MOV | 턴당 이동 가능 칸 수 |

- 레벨업 시 캐릭터별 고정 성장률에 따라 스탯이 자동 상승한다 (랜덤 요소 없음).
- MOV는 레벨업으로 오르지 않는다. 전직 또는 장비로만 변동.

### 3.3 스킬 습득

- 레벨 도달 시 자동 습득 (캐릭터별 습득 레벨 고정)
- 전직 시 상위 스킬 트리 해금
- 각 캐릭터는 **전직 전 스킬 4개 + 전직 후 스킬 4개** = 최대 8개 스킬 보유
- 전투 중 장착 가능한 액티브 스킬: 최대 **6개** (선택)
- 패시브 스킬: 항상 적용 (장착 불필요)

### 3.4 장비 시스템

슬롯 구성: **무기 1 + 방어구 1 + 악세서리 1**

| 슬롯 | 종류 | 효과 |
|------|------|------|
| 무기 | 검, 대검, 활, 단검, 지팡이, 성장, 창 등 | ATK/MATK 보정, 사거리, 특수 효과 |
| 방어구 | 경갑, 중갑, 로브, 망토 등 | DEF/MDEF/HP 보정 |
| 악세서리 | 반지, 목걸이, 부적, 부츠 등 | 다양한 보조 효과 (SPD+, 상태이상 면역, MOV+1 등) |

- 캐릭터별 장착 가능한 무기/방어구 타입이 제한된다.
- 악세서리는 전 캐릭터 공용.
- 전직 후 장착 가능 장비 범위가 확대되는 캐릭터도 있다.

## 6. 난이도 설계

### 6.1 2단계 난이도

| 항목 | Normal | Hard |
|------|--------|------|
| 적 HP | x1.0 | x1.3 |
| 적 ATK/MATK | x1.0 | x1.2 |
| 적 DEF/MDEF | x1.0 | x1.15 |
| 적 SPD | x1.0 | x1.1 |
| 적 AI | 기본 | 공격적, 집중 공격 |
| 아군 경험치 | x1.0 | x1.2 (보정) |
| 아이템 드롭 | 보통 | 약간 증가 |
| 부활 아이템 | 상점 구매 가능 (제한) | 이벤트 획득만 |

### 6.2 적 AI 차이

**Normal AI**:
- 가장 가까운 적을 우선 공격
- 체력 낮은 적 우선 공격 (기본 어그로)
- 힐러 유닛은 동료 치유 우선
- 유닛 간 협동 없음

**Hard AI**:
- 아군 힐러/마법사를 우선 노림 (위협도 기반)
- 집중 공격: 같은 대상에 여러 적이 몰림
- 적 힐러가 위치 선정을 더 잘함 (안전한 후방)
- 지형 활용: 숲/건물에 진을 치고 대기
- 우회: 탱커를 무시하고 후방 침투를 시도하는 적 유닛 존재

### 6.3 게임 오버

- 전투 패배 시 **해당 전투 시작 전으로 복귀** (경험치/아이템 유지 안 됨)
- Hard에서 연속 3회 패배 시 "난이도를 낮추시겠습니까?" 제안 (거부 가능)

│ [선택 유닛 정보] [대상 유닛 정보]       │
└──────────────────────────────────────┘
```

#### 유닛 선택 시

1. 유닛 클릭 → 이동 범위 표시 (파란 타일)
2. 이동 지점 클릭 → 유닛 이동 → 행동 메뉴 표시

#### 행동 메뉴

```
┌─────────┐
│ 공격     │ → 공격 범위 표시 (빨간 타일) → 대상 선택 → 데미지 예측 표시 → 확인
│ 스킬     │ → 스킬 목록 → 스킬 선택 → 범위 표시 → 대상 선택 → 확인
│ 아이템   │ → 아이템 목록 → 사용 → 대상 선택 (필요 시)
│ 대기     │ → 턴 종료
│ 방어     │ → 이동 안 했을 때만 표시
│ 취소     │ → 이동 전으로 되돌리기
└─────────┘
```

#### 정보 표시

- **유닛 위 HP 바**: 항상 표시
- **데미지 예측**: 공격/스킬 대상 선택 시 "예상 피해량 / 명중률 / 치명타율" 표시
- **지형 정보**: 커서를 놓은 타일의 지형 효과 하단에 표시
- **턴 순서 바**: 화면 상단에 다음 행동 유닛 순서 표시 (SPD 기준)

#### 전투 결과 화면

```

### 7.3 대화/이벤트 UI

- 하단 대화창: 캐릭터 초상화 + 이름 + 대사
- 선택지: 엔딩 분기 시점(4-8)에서만 등장. 일반 대화에는 선택지 없음.
- 이벤트 CG: 핵심 장면에서 전체 화면 일러스트 표시 (SD Forge로 생성)
  - 예: 피나가 빛에 흡수되는 장면, 세리아의 고백, 에리스의 죽음, 엔딩 CG
