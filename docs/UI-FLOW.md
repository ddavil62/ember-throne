## 1. 전체 흐름

```
[타이틀 화면]
    ├── 새 게임 → [난이도 선택] → [오프닝] → [월드맵]
    ├── 이어하기 → [세이브 선택] → [월드맵]
    └── 설정

[월드맵]
    ├── 스토리 거점 (아이콘) → [대화/이벤트] → [전투 준비] → [전투] → [결과] → [대화/이벤트] → [월드맵]
    ├── 상점 거점 → [상점 UI]
    ├── 유랑 전투 거점 → [전투 준비] → [전투] → [결과] → [월드맵]
    └── 메뉴
        ├── 편성 (유닛 선택, 장비 변경, 스킬 장착)
        ├── 아이템 (인벤토리)
        ├── 캐릭터 (스탯 확인, 관계도)
        ├── 도감 (적/아이템/스토리 회상)
        ├── 설정
        └── 세이브/로드
```

- 세이브 데이터가 없으면 "이어하기" 비활성

```
┌──────────────────────────────────────────────────────┐
│ [턴 순서 바: 다음 행동 유닛 아이콘 나열 (SPD 기준)]   │

│                                                      │

│         [이동 범위: 파란 타일]                         │
│         [공격 범위: 빨간 타일]                         │
│                                                      │
│                                                      │
│                                           [미니맵]   │
├──────────────────────────────────────────────────────┤
│ [선택 유닛 정보]          │  [지형 정보]              │
│ 이름: 카엘 Lv.12         │  숲 (회피+15%, 방어+5%)   │
│ HP: ████████░░ 72/100    │                           │
│ MP: ██████░░░░  45/80    │  [턴 표시]                │
│ 상태: 정상                │  플레이어 페이즈 - 턴 3   │
└──────────────────────────────────────────────────────┘
```

### 3.1 유닛 선택 시

1. 유닛 클릭 → 이동 범위 표시 (파란 타일)
2. 이동 지점 클릭 → 유닛 이동 → 행동 메뉴 표시

### 3.2 행동 메뉴

```
        ┌─────────────────┐
        │  공격            │ → 공격 범위 표시 (빨간 타일)
        │                 │   → 대상 선택 → 데미지 예측 표시 → 확인
        ├─────────────────┤
        │  스킬            │ → 스킬 목록 → 스킬 선택
        │                 │   → 범위 표시 → 대상 선택 → 확인
        ├─────────────────┤
        │  아이템          │ → 아이템 목록 → 사용
        │                 │   → 대상 선택 (필요 시)
        ├─────────────────┤
        │  대기            │ → 턴 종료
        ├─────────────────┤
        │  방어            │ → 이동 안 했을 때만 표시
        ├─────────────────┤
        │  취소            │ → 이동 전으로 되돌리기
        └─────────────────┘
```

### 3.3 데미지 예측 패널

공격/스킬 대상 선택 시 화면에 표시되는 예측 정보.

```
┌─────────────────────────────────────────────┐
│  공격자: 카엘 (Lv.12)    대상: 징발대 병사   │
│  무기: 강철 장검          무기: 철창          │
│  상성: 검 > 창 (유리!)                       │
├─────────────────────────────────────────────┤
│  예상 피해량:  28 ~ 34                       │
│  명중률:       87%                           │
│  치명타율:     12%                           │
│  반격 피해량:  8 ~ 12     (반격 명중률: 62%) │
└─────────────────────────────────────────────┘
```

### 3.4 정보 표시 규칙

- **유닛 위 HP 바**: 항상 표시
- **데미지 예측**: 공격/스킬 대상 선택 시 "예상 피해량 / 명중률 / 치명타율" 표시
- **지형 정보**: 커서를 놓은 타일의 지형 효과 하단에 표시
- **턴 순서 바**: 화면 상단에 다음 행동 유닛 순서 표시 (SPD 기준)
- **상성 표시**: 공격 시 유/불리가 공격자 옆에 화살표 아이콘으로 표시

## 4. 대화/이벤트 UI

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│              [배경 이미지 / 전투 맵]                   │
│                                                      │
│         [좌측 캐릭터]              [우측 캐릭터]       │
│         (발화자 밝게)              (비발화자 어둡게)    │
│                                                      │
├──────────────────────────────────────────────────────┤
│ ┌────────┐                                           │
│ │ 초상화  │  카엘                                     │
│ │        │  "왜 왕이 이런 명령을 내리는 거지...       │
│ │        │   알아내야 해."                            │
│ └────────┘                           [▼ 다음]        │
└──────────────────────────────────────────────────────┘
```

- 하단 대화창: 캐릭터 초상화 + 이름 + 대사
- 선택지: 엔딩 분기 시점(4-8)에서만 등장. 일반 대화에는 선택지 없음.
- 이벤트 CG: 핵심 장면에서 전체 화면 일러스트 표시 (SD Forge로 생성)
  - 예: 피나가 빛에 흡수되는 장면, 세리아의 고백, 에리스의 죽음, 엔딩 CG

## 5. 편성 화면 상세

### 5.1 편성 메인

월드맵 메뉴 또는 전투 준비에서 진입하는 편성 화면.

```
┌──────────────────────────────────────────────────────┐
│  [ 편성 ]                                 [ 닫기 ]   │
├────────────────────┬─────────────────────────────────┤
│                    │                                 │
│  출격 유닛 (8/8)    │      선택 유닛 상세              │
│  ┌──┬──┬──┬──┐    │  ┌────────┐                     │
│  │카엘│세리아│리넨│로크│  │ │ 초상화  │  카엘 Lv.12         │
│  │ * │    │    │    │    │ └────────┘  탈영기사             │
│  ├──┼──┼──┼──┤    │                                 │
│  │나엘│그리드│드라나│볼드│  │  HP: 100  MP: 45               │
│  │    │    │    │    │    │  ATK: 28  DEF: 22              │
│  └──┴──┴──┴──┘    │  MATK: 12  MDEF: 16             │
│                    │  SPD: 18  MOV: 4                │
│  대기 유닛          │                                 │
│  ┌──┬──┬──┬──┐    │  ──── 장비 ────                  │
│  │이렌│헤이즐│엘미라│(빈)│  │  무기:   녹슨 롱소드 [변경]     │

│                    │  악세: (없음)       [변경]       │

│  [유닛 입/퇴] 버튼  │  1. 강타          [변경]         │

│  ┌──────────────┐ │  3. (빈 슬롯)     [변경]         │
│  │ [장비] [스킬] │ │  4. (빈 슬롯)     [변경]         │
│  │ [아이템]      │ │                                 │
│  └──────────────┘ │                                 │
└────────────────────┴─────────────────────────────────┘
```

### 5.2 유닛 선택 (출격 편성)

```
조작 플로우:
1. 대기 유닛 목록에서 유닛 클릭
2. 출격 슬롯의 빈 자리 또는 교체할 유닛 클릭
3. 교체 확인 → 슬롯 갱신

규칙:
- 카엘(*)은 항상 출격 슬롯 1번에 고정
- 스토리 이벤트 고정 캐릭터도 제거 불가 (해당 맵 한정)
- 출격 슬롯: 최대 8칸
- 드래그 앤 드롭 또는 클릭-클릭 방식
```

### 5.3 장비 교체

```
┌──────────────────────────────────────────────────────┐
│  [ 장비 변경 - 무기 ]                      [ 닫기 ]   │
├──────────────────────┬───────────────────────────────┤
│                      │                               │
│  현재 장비            │  보유 무기 목록                │
│  ┌─────────────────┐ │  ┌───────────────────────────┐│
│  │ 녹슨 롱소드      │ │  │ 녹슨 롱소드     ATK+8    ││
│  │ ATK+8  사거리 1  │ │  │ ● 장착중               ││
│  │ (일반)           │ │  ├───────────────────────────┤│
│  └─────────────────┘ │  │ 강철 장검       ATK+14   ││
│                      │  │ 사거리 1 (고급)          ││
│  비교                 │  ├───────────────────────────┤│
│  ┌─────────────────┐ │  │ 은빛 레이피어   ATK+12   ││
│  │ ATK: 28 → 34    │ │  │ SPD+3  사거리 1 (고급)  ││
│  │ SPD: 18 → 18    │ │  ├───────────────────────────┤│
│  │ (▲ 상승 / ▼ 하락)│ │  │ ...                      ││
│  └─────────────────┘ │  └───────────────────────────┘│
│                      │                               │
│        [ 장착 ]       │  장착 불가 무기는 회색 처리    │
└──────────────────────┴───────────────────────────────┘
```

**플로우**:
1. 편성 상세에서 장비 슬롯의 [변경] 클릭
2. 좌측에 현재 장비, 우측에 보유 목록 표시
3. 목록에서 장비 선택 → 좌측에 스탯 비교 표시
4. [장착] 버튼으로 교체 확인
5. 캐릭터가 장착 불가한 유형은 회색 처리 + "장착 불가" 표시

```

│  │ 1. 강타          │ │  │ ● 강타        MP 8       ││
│  │ 2. 수호 자세     │ │  │   물리. 단일 대상 1.5배   ││
│  │ 3. (빈 슬롯)     │ │  ├───────────────────────────┤│
│  │ 4. (빈 슬롯)     │ │  │ ● 수호 자세    MP 12     ││

│                      │  │   직선 2칸 관통 공격       ││
│  ──── 패시브 ────    │  ├───────────────────────────┤│
│  [수호] 인접 아군     │  │ ...                      ││
│  피해 경감 10%       │  └───────────────────────────┘│
│  (항상 적용)          │                               │
│                      │  ● = 장착중, ○ = 미장착       │

└──────────────────────┴───────────────────────────────┘
```

**플로우**:
1. 편성 상세에서 스킬 슬롯의 [변경] 클릭
2. 좌측에 장착 슬롯(6칸), 우측에 보유 스킬 전체 목록
3. 스킬 선택 → 슬롯에 배치 (드래그 또는 클릭)
4. 패시브 스킬은 별도 영역에 상시 표시 (장착/해제 불가)
5. 미습득 스킬은 회색으로 표시 (습득 레벨 안내)
