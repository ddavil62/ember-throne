### 1.4 소재 아이템

재의 짐승 드롭 소재. 상점의 "소재 교환"에서 장비 제작에 사용한다.

| 소재 | 드롭처 | 교환 결과물 |
|------|--------|------------|
| 재의 이빨 | 재의 늑대 | 재의 단검 (고급 단검) |
| 재의 깃털 | 재의 독수리 | 바람의 망토 (고급 방어구) |
| 재의 거인 가죽 | 재의 거인 | 거인의 갑옷 (희귀 중갑) |
| 재의 수정 | 재의 마안 | 마안의 지팡이 (희귀 지팡이) |
| 재의 핵 | 재의 기생체 | 결계의 반지 (희귀 악세서리) |
| 재의 불씨 | 재의 불꽃 | 불꽃의 부적 (고급 악세서리) |

> 소재 교환에는 소재 + 골드가 필요하다. 교환은 각 거점 상점의 "대장간" 탭에서 수행한다.

## 2. 장비 — 무기

장비는 4등급 체계: **일반(Common) → 고급(Uncommon) → 희귀(Rare) → 유일(Unique)**.
유일 등급은 캐릭터 전용이며 스토리 이벤트로 획득한다.

## 부록 A: 등급별 스탯 범위 가이드

장비 설계 시 참고하는 등급별 스탯 범위.

| 등급 | ATK/MATK 범위 | DEF 범위 | 특수효과 |
|------|-------------|---------|---------|
| 일반(Common) | +3~+7 | +1~+6 | 없음 또는 미미 |
| 고급(Uncommon) | +8~+13 | +4~+10 | 단일 보정 (치명타, SPD 등) |
| 희귀(Rare) | +12~+17 | +8~+16 | 복합 보정 + 특수 기능 |
| 유일(Unique) | +12~+20 | +6~+18 | 캐릭터 전용 고유 효과 + 스킬 연동 |

> 유일 등급은 반드시 해당 캐릭터의 핵심 스킬과 시너지를 가진다. 수치 자체는 희귀급 상한이지만, 고유 효과가 압도적 가치를 지닌다.

## 부록 B: 아이템 인벤토리 제한

| 항목 | 제한 |
|------|------|
| 소비 아이템 종류 | 무제한 |
| 소비 아이템 개당 소지 수 | 최대 20개 |
| 장비 아이템 총 보관 | 최대 100개 (장착 중 포함) |
| 핵심 아이템 | 별도 탭, 제한 없음 |
| 소재 아이템 | 별도 탭, 개당 최대 99개 |

> 장비 보관이 100개를 초과하면 상점에서 판매를 유도하는 알림이 표시된다.
