## 게임 개요

| 항목 | 내용 |
|------|------|
| 제목 | **Ember Throne** |
| 장르 | 택티컬 RPG (그리드 기반 턴제) |
| 플랫폼 | Steam (Windows / Linux / macOS) |
| 엔진 | Godot 4.6.2 (GDScript) |
| 아트 스타일 | 도트 스프라이트 (캐릭터 48px, 타일 32px) + SD/PixelLab 파이프라인 |

### 핵심 컨셉

- **그리드 기반 턴제 전투**: 플레이어 페이즈 -> 적 페이즈 구조. 지형 효과와 무기 상성으로 전술적 깊이 확보.
- **12인의 고유 캐릭터**: 각각 1:1 대응 클래스, 스토리 연동 전직, 캐릭터 서사와 전투 역할이 연결됨.
- **정치극 서사**: 시원화(始原火)의 쇠퇴를 둘러싼 중앙 vs 변방의 갈등. 3개 러브라인 + 4개 비극 서사.
- **멀티 엔딩**: 엔딩 A (시원화 재점화) / 엔딩 B (불꽃의 분산) -- 최종전 직전 선택.
- **최대 출격 8명**: 12명 중 8명 선택 편성, 카엘 고정 출격.
- **전투 35전**: 필수 28전 + 선택 7전 (유랑 전투).
- **난이도 2단계**: Normal / Hard (적 스탯/AI 차이 중심).

## 부록: 레퍼런스 매핑

| 요소 | 레퍼런스 | 차용/변형 |
|------|----------|-----------|
| 턴 구조 | 파랜드 택틱스 | 플레이어 전체 -> 적 전체 (동일) |
| 클래스/전직 | 파이어 엠블렘 | 스토리 연동 자동 전직 (FE는 레벨 기반) |
| 무기 상성 | 파이어 엠블렘 | 검>도끼>창>검 삼각 + 마법 상성 |
| 지형 효과 | 파이어 엠블렘, 택틱스 오우거 | 숲 회피 보정, 결계석/재의 땅 추가 |
| 정치 서사 | 창세기전 2/3 | 왕가 비극, 대의와 개인의 갈등 |
| 캐릭터 유대 | FE 창성 | 인접 보정, 쌍방 패시브 |
| 멀티 엔딩 | 택틱스 오우거 | 2분기 (시원화 재점화 / 불꽃 분산) |
| 출격 인원 | 파랜드 택틱스 2 | 8명 출격, 12명 중 선택 |
| 난이도 | FE 시리즈 | 2단계, AI 차이 중심 |
| 경험치 보정 | FE 시리즈 | 적-아군 레벨 차이에 따른 보정 계수 |
