# Ember Throne - 전투 시스템

> 초안 v0.1 | 2026-04-06
> 상위 문서: [GAME-DESIGN.md](GAME-DESIGN.md)

---

## 1. 기본 구조

그리드 기반 턴제 전투. **플레이어 페이즈 -> 적 페이즈** 순서로 진행된다.

- 플레이어 페이즈: 아군 전 유닛을 자유 순서로 이동/행동할 수 있다. 모든 유닛의 행동이 끝나거나 "페이즈 종료"를 선택하면 적 페이즈로 넘어간다.
- 적 페이즈: 적 AI가 모든 적 유닛을 순차 행동시킨다.
- NPC 페이즈: 호위 대상, 중립 유닛이 존재하는 맵에서만 발생한다.

## 2. 유닛 행동

각 유닛은 턴당 **이동 1회 + 행동 1회**를 수행할 수 있다.

| 행동 | 설명 |
|------|------|
| 공격 | 무기 사거리 내 적에게 물리 공격 |
| 스킬 | MP를 소모하여 고유 스킬 사용 |
| 아이템 | 소비 아이템 사용 (자신 또는 인접 아군) |
| 대기 | 행동 없이 턴 종료. 방어 보정 +10% |
| 방어 | 이동 포기 시 선택 가능. 받는 피해 -30%, 방어 보정 +20% |

- 이동 후 행동, 행동 후 이동 모두 가능하다 (단, 각 1회씩).
- "이동만 하고 행동 안 함"도 가능하다 (남은 행동은 소멸).

## 3. 이동 범위와 공격 범위

- **이동 범위**: 유닛의 MOV 스탯에 의해 결정. 맨해튼 거리(다이아몬드 형태) 기준.
- **공격 범위**: 무기 유형에 따라 다름.

| 무기 유형 | 기본 사거리 | 비고 |
|-----------|------------|------|
| 검/도끼/창 | 1칸 | 인접 4방향 |
| 창(장창) | 1~2칸 | 관통 불가 |
| 활 | 2~3칸 | 인접 불가 (최소 사거리 2) |
| 마법 | 1~3칸 (스킬별 상이) | 스킬마다 범위/형태 다름 |
| 단검 | 1칸 | 인접 4방향, 후방 공격 시 치명타 보정 |
| 지팡이(타격) | 1칸 | 긴급용, 낮은 위력 |

## 4. 지형 효과

맵의 각 타일에는 지형 유형이 지정되며, 유닛에게 영향을 준다.

| 지형 | 이동 비용 | 회피 보정 | 방어 보정 | 특수 효과 |
|------|-----------|-----------|-----------|-----------|
| 평지 | 1 | 0% | 0% | - |
| 숲 | 2 | +15% | +5% | 활 사거리 -1 |
| 산/바위 | 3 | +10% | +10% | 중장 유닛 진입 불가 |
| 물(얕은) | 2 | -5% | -5% | 마법 공격 +10% |
| 물(깊은) | 진입 불가 | - | - | 특수 스킬로만 이동 |
| 성벽/건물 | 진입 불가 | - | - | 파괴 가능한 건물 존재 |
| 폐허 | 1 | +5% | 0% | - |
| 모래 | 2 | 0% | 0% | 기마 유닛 MOV -1 |
| 다리 | 1 | 0% | 0% | 파괴 가능 |
| 결계석 | 1 | 0% | 0% | 인접 아군 턴 시작 시 HP 5% 회복 |
| 재의 땅 | 2 | 0% | -10% | 턴 시작 시 HP 5% 감소 |

## 5. 높이 시스템

본작은 **높이차 개념을 도입하지 않는다.** 탑다운 2D 그리드를 유지하며, 지형 효과로 전술적 다양성을 확보한다. 산/바위는 높이 대신 "진입 불가 또는 높은 이동 비용"으로 표현한다.

## 6. 전투 공식

```
물리 데미지 = (공격자 ATK * 스킬 배율) - (방어자 DEF / 2)
마법 데미지 = (공격자 MATK * 스킬 배율) - (방어자 MDEF / 2)
최종 데미지 = 기본 데미지 * 지형 보정 * 상성 보정 * (1 + 치명타 보정)
명중률 = 공격자 명중 - 방어자 회피 - 지형 회피 보정
치명타율 = 기본 치명타율 + (SPD차 / 4)
```

- 최소 데미지: 1 (0 이하 방지)
- 명중률 범위: 5%~99% (확정 명중/회피 없음)
- 치명타 데미지: 기본 데미지 * 1.5

## 7. 무기 상성 시스템 (Weapon Triangle)

파이어 엠블렘 시리즈를 레퍼런스로, 무기 유형 간 상성 삼각을 도입한다. 상성 유리 시 명중/데미지가 보정되고, 불리 시 감소한다.

### 7.1 상성 관계

```
검 (Sword) ──▶ 도끼 (Axe) ──▶ 창 (Lance) ──▶ 검 (Sword)
    ↑               ↑               ↑
    └───────────────┴───────────────┘  (역방향 불리)
```

- **검 > 도끼**: 검의 속도가 도끼의 둔중함을 이긴다
- **도끼 > 창**: 도끼의 파괴력이 창의 리치를 압도한다
- **창 > 검**: 창의 사거리가 검의 접근을 차단한다

### 7.2 상성 보정 수치

| 상성 관계 | 명중 보정 | 데미지 보정 |
|-----------|-----------|-------------|
| 유리 | +15% | +15% |
| 불리 | -15% | -15% |
| 중립 (동일 무기 또는 비해당) | 0% | 0% |

### 7.3 마법 상성 (부차적)

마법 유형 간에도 약한 상성이 존재한다. 물리 상성보다 보정치가 낮다.

```
화염 (Fire) ──▶ 풍 (Wind) ──▶ 뇌 (Thunder) ──▶ 화염 (Fire)
빛 (Light) ◀──▶ 어둠 (Dark)  (상호 유리)
```

| 상성 관계 | 명중 보정 | 데미지 보정 |
|-----------|-----------|-------------|
| 유리 | +10% | +10% |
| 불리 | -10% | -10% |

### 7.4 상성과 캐릭터 매핑

| 무기 유형 | 사용 캐릭터 |
|-----------|-------------|
| 검 | 카엘, 볼드(검+방패) |
| 대검 | 헤이즐 |
| 도끼 | - (적 전용, 아군 도끼 캐릭터 없음) |
| 창 | - (적 전용) |
| 단검 | 리넨, 그리드, 이렌 |
| 활 | 로크 |
| 지팡이/성장 | 세리아, 나엘, 드라나, 엘미라 |

- **단검, 활, 지팡이/성장**은 상성 삼각에 포함되지 않는다 (중립).
- 대검은 검 카테고리에 포함되나, 무게 패널티로 SPD -2 보정.
- 적 진영에는 도끼/창 무기 유닛이 다수 등장하므로, 아군 검 유닛(카엘, 볼드, 헤이즐)은 상성에서 다양한 유불리를 경험한다.

## 8. 상태이상

| 상태 | 효과 | 지속 | 회복 |
|------|------|------|------|
| 독 | 턴 시작 시 최대 HP 8% 감소 | 3턴 | 해독제, 치유 스킬 |
| 화상 | 턴 시작 시 최대 HP 5% 감소, DEF -15% | 2턴 | 자연 치유, 물 지형 진입 시 즉시 해제 |
| 빙결 | 행동 불가 | 1턴 | 피격 시 해제, 화염 스킬 해제 |
| 마비 | 이동 불가, 행동은 가능 | 2턴 | 회복 아이템, 치유 스킬 |
| 침묵 | 스킬 사용 불가 | 2턴 | 자연 치유, 회복 아이템 |
| 수면 | 행동 불가, 피격 시 해제 | 3턴 (피격 해제) | 피격, 회복 아이템 |
| 재화(灰化) | 턴 시작 시 최대 HP 10% 감소, 모든 스탯 -10% | 3턴 | 결계석 인접 시 즉시 해제, 상급 치유 |

"재화"는 재의 바다 관련 적이 사용하는 고유 상태이상으로, 스토리 후반에 등장한다.

## 9. 승리/패배 조건

- **기본 승리**: 적 전멸
- **기본 패배**: 카엘(주인공) 전투불능
- **특수 조건**: 맵별로 추가 조건 설정 가능
  - 방어전: N턴 생존
  - 호위전: 특정 NPC 생존
  - 탈출전: 카엘이 탈출 지점 도달
  - 보스전: 보스 유닛 격파 (잡졸 전멸 불필요)

## 10. 전투 흐름 상세 (Turn Flow)

전투의 한 턴은 아래 플로우차트에 따라 진행된다.

### 10.1 턴 시작 ~ 종료 플로우

```
[ 턴 시작 ]
    │
    ├── 턴 카운터 증가
    ├── 지형 효과 적용 (재의 땅 HP 감소, 결계석 HP 회복 등)
    ├── 상태이상 틱 적용 (독/화상/재화 데미지, 빙결/마비/침묵/수면 지속 카운트 -1)
    ├── 버프/디버프 지속 카운트 -1 (만료 시 제거)
    │
    ▼
[ 플레이어 페이즈 ]
    │
    ├── 행동 가능한 아군 유닛 하이라이트
    ├── 유닛 선택 대기 (자유 순서)
    │     │
    │     ▼
    │   [ 유닛 선택 ]
    │     │
    │     ├── 이동 범위 표시 (파란 타일)
    │     ├── 이동 지점 선택 → 유닛 이동 애니메이션
    │     │
    │     ▼
    │   [ 행동 메뉴 ]
    │     │
    │     ├── [공격] → 공격 범위 표시 (빨간 타일)
    │     │            → 대상 선택 → 상성/데미지 예측 표시
    │     │            → 확인 → 전투 애니메이션 → 결과 처리
    │     │
    │     ├── [스킬] → 스킬 목록 표시 (MP 부족 시 비활성)
    │     │            → 스킬 선택 → 범위 표시
    │     │            → 대상 선택 → 예측 표시
    │     │            → 확인 → 스킬 연출 → 결과 처리
    │     │
    │     ├── [아이템] → 소지 아이템 목록
    │     │              → 아이템 선택 → 대상 선택 (필요 시)
    │     │              → 사용 → 효과 적용
    │     │
    │     ├── [대기] → 방어 보정 +10% 부여 → 유닛 행동 완료
    │     │
    │     ├── [방어] → (이동 안 한 경우만) 피해 -30%, 방어 보정 +20% → 유닛 행동 완료
    │     │
    │     └── [취소] → 이동 전 위치로 복귀 → 유닛 선택 대기로 돌아감
    │
    │   ※ 유닛 행동 완료 → 유닛 그레이아웃 → 다음 유닛 선택 대기
    │
    ├── 모든 아군 행동 완료 OR "페이즈 종료" 선택
    │
    ▼
[ 적 페이즈 ]
    │
    ├── 적 AI가 유닛별 행동 결정 (난이도별 AI 로직 참조: DIFFICULTY.md)
    ├── 각 적 유닛 순차 행동 (이동 → 행동 → 완료)
    ├── 모든 적 행동 완료
    │
    ▼
[ NPC 페이즈 ] (해당 맵에 NPC가 있을 때만)
    │
    ├── NPC 유닛 AI 행동 (호위 대상: 도주/방어, 중립: 독자 행동)
    ├── 모든 NPC 행동 완료
    │
    ▼
[ 턴 종료 ]
    │
    ├── 승리/패배 조건 체크
    │     ├── 승리 조건 충족 → [ 전투 결과 화면 ]
    │     ├── 패배 조건 충족 → [ 게임 오버 처리 ]
    │     └── 미충족 → 다음 턴 시작으로
    │
    └── → [ 다음 턴 시작 ]
```

### 10.2 전투 전 준비 플로우

```
[ 전투 선택 (월드맵/스토리) ]
    │
    ▼
[ 전투 준비 화면 ]
    │
    ├── 출격 유닛 편성 (최대 8명, 카엘 필수)
    ├── 장비 변경 (무기/방어구/악세서리)
    ├── 스킬 장착 변경 (액티브 최대 6개)
    ├── 아이템 배분
    │
    ▼
[ 맵 미리보기 ]
    │
    ├── 맵 전체 구조 확인
    ├── 이렌 "정보망" 패시브 보유 시: 적 배치 공개
    ├── 아군 초기 배치 지점 선택
    │
    ▼
[ 전투 개시 ]
```

### 10.3 경험치 공식

경험치는 전투 행동마다 획득되며, 적 레벨과의 차이에 따라 보정된다.

**기본 경험치 공식**:

```
행동 경험치 = 기본값 * 레벨 보정 계수
적 처치 보너스 = 기본 처치 경험치 * 레벨 보정 계수

기본값:
  - 공격(명중 시): 10
  - 공격(미스): 1
  - 치유 스킬 사용: 10
  - 버프/디버프 스킬 사용: 8
  - 아이템 사용: 3

기본 처치 경험치: 30
```

**레벨 보정 계수 (적 레벨 - 아군 레벨 = diff)**:

| diff | 보정 계수 | 설명 |
|------|-----------|------|
| +5 이상 | 2.0 | 아군보다 5레벨 이상 높은 적 |
| +4 | 1.8 | |
| +3 | 1.6 | |
| +2 | 1.4 | |
| +1 | 1.2 | |
| 0 | 1.0 | 동일 레벨 |
| -1 | 0.85 | |
| -2 | 0.70 | |
| -3 | 0.55 | |
| -4 | 0.40 | |
| -5 이하 | 0.25 | 아군보다 5레벨 이상 낮은 적 (최소 보정) |

**레벨업 필요 경험치**:

```
필요 경험치 = 100 * 현재 레벨
예: Lv.1 → Lv.2 = 100 EXP, Lv.10 → Lv.11 = 1000 EXP
```

**비참가 유닛 보정**:

- 전투 미참가 유닛은 전투 종료 시 참가 유닛 평균 획득 경험치의 **50%**를 획득한다.
- 이를 통해 벤치 유닛의 레벨 격차가 과도하게 벌어지는 것을 방지한다.
