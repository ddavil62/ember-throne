## 1. 2단계 난이도

| 항목 | Normal | Hard |
|------|--------|------|
| 적 HP | x1.0 | x1.3 |
| 적 ATK/MATK | x1.0 | x1.2 |
| 적 DEF/MDEF | x1.0 | x1.15 |
| 적 SPD | x1.0 | x1.1 |
| 적 AI | 기본 | 공격적, 집중 공격 |
| 아군 경험치 | x1.0 | x1.2 (보정) |
| 아이템 드롭 | 보통 | 약간 증가 |
| 부활 아이템 | 상점 구매 가능 (제한) | 이벤트 획득만 |

## 2. 적 AI 차이

**Normal AI**:
- 가장 가까운 적을 우선 공격
- 체력 낮은 적 우선 공격 (기본 어그로)
- 힐러 유닛은 동료 치유 우선
- 유닛 간 협동 없음

**Hard AI**:
- 아군 힐러/마법사를 우선 노림 (위협도 기반)
- 집중 공격: 같은 대상에 여러 적이 몰림
- 적 힐러가 위치 선정을 더 잘함 (안전한 후방)
- 지형 활용: 숲/건물에 진을 치고 대기
- 우회: 탱커를 무시하고 후방 침투를 시도하는 적 유닛 존재

## 3. 게임 오버

- 전투 패배 시 **해당 전투 시작 전으로 복귀** (경험치/아이템 유지 안 됨)
- Hard에서 연속 3회 패배 시 "난이도를 낮추시겠습니까?" 제안 (거부 가능)
