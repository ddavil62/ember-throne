## 1. 스킬 연출 등급 체계

스킬을 연출 복잡도에 따라 4등급으로 분류한다.

| 등급 | 명칭 | 연출 시간 | 구성 요소 | 사용 빈도 |
|------|------|-----------|-----------|-----------|
| **S** | 궁극기(Ultimate) | 3~5초 | 컷인 -> 전용 애니메이션 -> 전체화면 이펙트 -> 히트 | 전투당 0~1회 |
| **A** | 상급 스킬 | 2~3초 | 캐스팅 -> 전용 이펙트 -> 히트 이펙트 | 전투당 2~4회 |
| **B** | 일반 스킬 | 1~2초 | 짧은 캐스팅 -> 공용 이펙트(색상/형태 변형) -> 히트 | 자주 사용 |
| **C** | 기본 행동 | 0.5~1초 | 무기 스윙/사격 모션 -> 히트 | 매 턴 |

### 1.1 등급 배정 원칙

- **S급**: 각 캐릭터의 최종 스킬 (Lv25~26 궁극기) -- 11개
- **A급**: 전직 후 해금되는 강력한 스킬 -- 중상위 스킬
- **B급**: 전직 전 중~하위 스킬, 전직 후 유틸리티 스킬
- **C급**: 기본 공격, 대기, 방어, 아이템 사용

## 2. 컷인 시스템 (Cut-in System)

궁극기(S급) 사용 시 발동하는 연출 시스템.

### 2.1 컷인 플로우

```
[유닛 스킬 선택] -> [화면 암전 (0.3초)]
    -> [컷인 초상화 슬라이드 인 (0.5초)]
        - 캐릭터 반신 일러스트가 화면 좌측에서 빠르게 슬라이드
        - 배경: 캐릭터 테마 컬러 집중선(스피드라인) 효과
        - 스킬명 텍스트가 우측에 대각선으로 출현
    -> [카메라 줌아웃 (0.3초)]
        - 전투 맵으로 복귀
    -> [스킬 전용 애니메이션 (1.5~2초)]
        - 시전자 특수 모션
        - 범위 이펙트
        - 히트 이펙트 + 데미지 넘버
    -> [결과 표시]
```

### 2.2 컷인 아트 사양

| 항목 | 사양 |
|------|------|
| 해상도 | 512x768 (세로형 반신) |
| 아트 스타일 | SD Forge SDXL로 생성, 모던 판타지 컨셉아트 기반 |
| 표정 | 캐릭터별 궁극기 사용 시의 감정 (결의, 분노, 각오 등) |
| 포즈 | 스킬 시전 포즈 (검 휘두르기, 마법 시전, 방패 들기 등) |
| 배경 | 투명 (런타임에 집중선 효과 오버레이) |
| 장수 | 11장 (플레이어블 캐릭터 전원) |
| 제작 파이프라인 | SD Forge SDXL -> 배경 제거 -> Godot Sprite2D |

### 2.3 컷인 연출 상세

캐릭터별 컷인 테마:

| 캐릭터 | 궁극기 | 컷인 테마 컬러 | 표정/포즈 | 집중선 스타일 |
|--------|--------|---------------|-----------|-------------|
| 카엘 | 새벽의 검 | 붉은 금색 (ember) | 결의에 찬 얼굴, 검을 높이 들어올림 | 방사형 금빛 집중선 |
| 세리아 | 사도의 축복 | 순백+금 | 눈 감고 기도, 빛 후광 | 수직 하강 빛줄기 |
| 리넨 | 계승자의 의지 | 주황+회색 잿빛 | 양손을 펼쳐 화염 해방, 눈에 불꽃 | 소용돌이형 화염 |
| 로크 | 수호자의 일제 | 숲 녹색 | 활을 하늘로 겨냥, 바람 | 대각선 화살 궤적 |
| 나엘 | 유대의 공명 | 연보라+핑크 | 양팔 벌려 노래, 음표 이펙트 | 파동형 동심원 |
| 그리드 | 부친의 맹세 | 짙은 남색+번개 | 이를 악물고 곡검 돌격 자세 | 직선 돌진 스피드라인 |
| 드라나 | 렌도르의 유산 | 보라+백색 전기 | 마법진 위에 부유, 두 손 모아 캐스팅 | 복잡한 마법진 회전 |
| 볼드 | 최후의 방벽 | 강철색+황금 | 방패 정면으로, 부서지면서 빛나는 | 파편 방사형 |
| 이렌 | 총지휘 | 다크 네이비 | 코트 날리며 지휘봉 치켜듬 | 수평 스피드라인 |
| 헤이즐 | 해방의 깃발 | 진홍 | 대검 땅에 꽂으며 포효 | 수직 충격파 |
| 엘미라 | 불꽃 의식 | 백금+화염 | 성장 들고 신성한 의식 자세 | 원형 빛 폭발 |

### 2.4 Godot 구현 컨셉

```
CutInSystem (Node2D)
+-- DimOverlay (ColorRect)         -- 암전 레이어
+-- SpeedLines (AnimatedSprite2D)  -- 집중선
+-- PortraitSprite (Sprite2D)      -- 컷인 초상화
+-- SkillNameLabel (Label)         -- 스킬명 텍스트
+-- AnimationPlayer                -- 타이밍 제어
```

- 컷인은 싱글톤 또는 BattleUI 자식으로 관리
- AnimationPlayer로 시퀀스 제어 (암전 -> 슬라이드 -> 줌아웃)
- 스킬명 폰트: 붓글씨/캘리그라피 느낌 (판타지 분위기)
- 컷인 스킵: 설정에서 "궁극기 연출 스킵" 옵션 제공

### 3.3 렌더링 방식

| 방식 | 설명 | Godot 구현 | 장단점 |
|------|------|------------|--------|
| **스프라이트 애니메이션** | 미리 그린 프레임 시퀀스 | AnimatedSprite2D | 도트감 유지, 제작 비용 높음 |
| **파티클 시스템** | Godot GPUParticles2D | GPUParticles2D | 가변적, 가볍고, 도트감 약함 |
| **하이브리드** | 스프라이트 + 파티클 혼합 | 복합 씬 | 최적, 궁극기에 사용 |
| **셰이더** | 화면 효과 (글로우, 흔들림 등) | ShaderMaterial | 연출 보강용 |
| **프로젝타일** | 비행체 궤적 | Area2D + Tween | 화살, 마법탄 등 |

## 6. 연출 규칙

### 6.1 컷인 발동 조건

- **컷인은 S급(궁극기)에서만 발동한다** -- A급 이하에서는 발동하지 않음.
- **같은 전투에서 같은 캐릭터의 컷인은 최초 1회만** 풀 재생, 이후는 축약(0.5초 축소판).
- **설정에서 "연출 스킵" 옵션 제공** -- 컷인 OFF, 이펙트 간소화 선택 가능.
- **적 보스도 컷인 있음**: 변질 모르간 페이즈 전환 시 적 컷인 (1장).

### 6.2 히트 이펙트 스케일링

- **히트 이펙트 크기는 데미지에 비례**: 약한 히트는 작은 이펙트, 강한 히트는 크고 화면 셰이크.
- 데미지 구간별 이펙트 스케일:

| 데미지 구간 | 이펙트 스케일 | 화면 셰이크 | 넘버 크기 |
|-------------|-------------|-------------|-----------|
| 1~20 | 0.8x | 없음 | 소 |
| 21~50 | 1.0x | 없음 | 중 |
| 51~100 | 1.3x | 미세 (1px, 0.1초) | 대 |
| 101~150 | 1.5x | 중 (2px, 0.2초) | 대+강조색 |
| 151+ | 2.0x | 강 (3px, 0.3초) | 최대+빛줄기 |
| 치명타 | 추가 +0.3x | 추가 셰이크 | 황금색 |

### 6.3 범위 공격 시차 연출

- 범위 공격 시 히트 이펙트는 **시차 발생** (0.1초 간격).
- 순서: 시전자에서 **가까운 타일 -> 먼 타일** (바깥->안쪽이 아닌, 중심->외곽).
- 원형 범위: 중심 -> 인접 -> 외곽 순서.
- 직선 범위: 시전자 가까운 쪽 -> 먼 쪽 순서.

### 6.4 동시 이펙트 제한

- **최대 동시 파티클 이미터**: 5개.
- **최대 총 입자 수**: 200개 (PC 타겟이므로 여유롭지만 상한선 설정).
- 범위 공격의 히트 이펙트가 5개를 초과하면 시차를 0.15초로 확대하여 순차 재생.

### 6.5 사운드 연출 동기화

- 모든 이펙트는 SE(효과음)와 동기화된다.
- 컷인 시작 시: 전용 SE (캐릭터별 차별화).
- 히트 순간: 히트 SE (물리/마법/치유 분류).
- 화면 셰이크 시: 중저음 임팩트 SE.

### 6.6 특수 연출 상세

| 조건 | 추가 연출 | 설명 |
|------|-----------|------|
| 전투불능 처리 | 유닛 페이드아웃 (0.5초) | 피격 -> HP 0 -> 쓰러짐 모션 -> 투명 페이드 |
| 레벨업 | FX_LIGHT_PILLAR + SE | 전투 종료 후 결과 화면에서 레벨업 연출 |
| 상태이상 부여 | 대상 위 아이콘 팝업 | 독(초록), 화상(주황), 빙결(파랑), 마비(노랑), 침묵(보라), 수면(파랑), 재화(회색) |
| 상태이상 해제 | 아이콘 파쇄 이펙트 | 해당 아이콘이 깨지며 사라짐 |
| 버프/디버프 부여 | 대상에 오라 / 균열 이펙트 | 지속 시간 동안 유지, 만료 시 페이드아웃 |
| 넉백 | 대상 강제 이동 애니메이션 | 볼드 돌진 등, 1칸 밀려남 (0.2초) |
| 도발 | 적 유닛 위 도발 마커 | 빨간 느낌표 아이콘, 2턴간 유지 |
| 카엘 서약의 유대 발동 | 피해 대신 받기 연출 | 카엘이 대상 위치로 이동 -> 피격 -> HP 1 잔존, FX_SHIELD_GLOW |
| 볼드 최후의 방벽 발동 | 컷인 + 불사 연출 | HP 10% 이하 도달 시 자동 컷인, 이후 3턴간 HP 바 빛남 |

## 부록 B: 패시브 스킬 시각 표현

패시브 스킬은 전투 중 발동 시 간략한 시각적 피드백을 제공한다.

| 패시브 | 캐릭터 | 발동 조건 | 시각 표현 |
|--------|--------|-----------|-----------|
| 탈영기사의 긍지 | 카엘 | 인접 아군 피격 시 | 카엘에게 순간 FX_SHIELD_GLOW (미세) |
| 서약의 유대 | 카엘 | 맹세 대상 위기 시 | 카엘 이동 연출 + 피격 대신 받기 |
| 이단의 신앙 | 세리아 | 치유 스킬 사용 시 | 치유 이펙트에 추가 반짝임 |
| 신앙의 불꽃 | 세리아 | 치유 시 | 대상에 미세 FX_BUFF_AURA (1턴) |
| 방랑자의 직감 | 리넨 | 상시 / 함정 감지 시 | 함정 인접 시 느낌표 아이콘 |
| 불꽃 공명 (패시브) | 리넨 | 결계석 타일 위 | 리넨 유닛 글로우 오버레이 |
| 산림 주파 | 로크 | 숲 지형 이동 시 | 이동 경로에 녹색 잎 파티클 |
| 선제 사격 | 로크 | 적 이동 시 반격 | 자동 RANGED_SHOOT + FX_ARROW_SINGLE |
| 야전 군의관 | 나엘 | 아이템 사용 시 | 아이템 이펙트에 추가 반짝임 |
| 유대 | 나엘 | 리넨 인접 시 | 쌍방 유닛에 연결선 이펙트 (연보라) |
| 생명의 오라 | 나엘 | 매 턴 시작 시 | 인접 아군에 미세 FX_HEAL_SPARKLE |
| 밀수업자의 눈 | 그리드 | 맵 시작 시 | 보물상자 위치 하이라이트 |
| 마리나의 약속 | 그리드 | HP 20% 이하 피격 시 | FX_SHIELD_GLOW (순간) + 피해 무효 팝업 |
| 수석의 집중 | 드라나 | 마법 사용 시 | 시전 시 드라나 주위 미세 마법진 |
| 렌도르의 유산 (연속 시전) | 드라나 | 마법 연속 사용 시 | MP 소모 감소 넘버 색상 변경 (파란색) |
| 근위대장의 위엄 | 볼드 | 도발 사용 시 | 도발 대상 적 ATK 감소 아이콘 |

| 정보망 | 이렌 | 맵 시작 시 | 적 유닛 이동 범위 표시 (1턴) |
| 복수의 끝 | 이렌 | 보스 공격 시 | 히트 이펙트 색상 변경 (진홍) |
| 장군의 위엄 | 헤이즐 | 상시 (인접 아군) | 인접 아군에 미세 FX_BUFF_AURA |
| 양심의 칼 | 헤이즐 | 카엘 인접 시 | 쌍방 유닛에 연결선 이펙트 (황금) |
| 대사제의 가호 | 엘미라 | 치유/마법 사용 시 | 시전 시 성장에서 빛 발산 |
