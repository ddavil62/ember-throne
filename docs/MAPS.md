## 6. 유랑 전투 (선택 전투) 상세

유랑 전투는 월드맵에서 선택적으로 진입할 수 있는 레벨링/아이템 파밍용 전투다. 스토리 진행에 필수는 아니지만, Hard 난이도에서는 레벨 보충에 유용하다.

### 6.1 유랑 전투 1: 산적 소탕

| 해금 시점 | 1막 중 (1-7 이후) |
| 맵 | 가도(소) 15x12 |
| 지형 | 평지 60%, 숲 20%, 폐허 10%, 바위 10% |
| 적 구성 | 산적 두목 Lv.4 x1, 산적 보병 Lv.3 x3, 산적 궁수 Lv.3 x2 |
